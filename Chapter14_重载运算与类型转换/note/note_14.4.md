# 14.4 赋值运算符

重载的赋值运算符必须释放当前内存空间，再创建一片新空间。

赋值运算符都必须定义为成员函数。

复合赋值运算符也应该定义为成员函数。

两类赋值运算符都应该返回左侧运算对象的引用。
