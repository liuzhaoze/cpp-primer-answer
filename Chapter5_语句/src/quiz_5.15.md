说明下列循环的含义并改正其中的错误。

(a)

```c
for (int ix = 0; ix != sz; ++ix) { /* ... */ }
if (ix != sz)
    // ...
```

(b)

```c
int ix;
for (ix != sz; ++ix) { /* ... */ }
```

(c)

```c
for (int ix = 0; ix != sz; ++ix, ++sz) { /* ... */ }
```

> 答：  
> (a) 变量 ix 在循环体外不可见。  
> (b) for 语句头缺少一条语句，应该在最前面加上 “;” 。  
> (c) sz 是迭代对象的长度，不应该对其进行增加。
